Portafolio de Inversiones

Este proyecto es una aplicación web interactiva diseñada para registrar y visualizar inversiones y activos financieros. Su desarrollo forma parte de mi proceso de aprendizaje continuo en programación y desarrollo web, sirviendo como una demostración de habilidades técnicas en la integración de frontend, backend y bases de datos.

⚠️ Nota: Este proyecto no es un producto final para uso en producción. Es un proyecto de práctica y un portafolio de 
aprendizaje.🚀 (tiene detalle que pulir)

Características
Registro de usuario: Carga de inversiones personales de forma dinámica.
Gestión de activos: Panel para visualizar diferentes tipos de activos financieros.
Interfaz limpia: Diseño estructurado para una visualización clara de tablas de datos.
Persistencia: Conexión con base de datos relacional para almacenar toda la información.

🛠️ Tecnologías Utilizadas
Backend: Python (con Flask)
Frontend: HTML5, CSS3, JavaScript
Base de Datos: SQLite (database.db)Control de Versiones: Git y GitHub

Durante el desarrollo de este proyecto, reforcé conocimientos en:
Conectar una aplicación backend con una base de datos SQL.
Renderizar datos dinámicos desde el servidor a plantillas HTML.
Organizar y estructurar un proyecto siguiendo el patrón adecuado para aplicaciones web.
